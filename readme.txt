Jogo de 2 Jogadores

Objetivo é desviar dos meteoros e ter a maior pontuação após 30s. A pontuação inicial é 1000 pontos e o jogador perde -50 pontos por ser
acertado por um meteoro. O jogo acaba quando um jogador atinge 0 pontos ou após bater o tempo de 30s. Se ambos jogadores tiverem a mesma
pontuação no final do jogo, é dado Empate.
